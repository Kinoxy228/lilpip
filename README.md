# VD Games Project

Это проект для игр от lilpip (Padzhev). Содержит консольную/графическую игру, запускаемую командой `VD-games`.

## Установка
1. Клонируйте репозиторий.
2. `make install` для зависимостей.
3. `make VD-games` для запуска.

## Зависимости
- Python >3.12

## Запуск
`VD-games`
